## [0.1.0] - 2025-11-21

### Added

- Initial public release
- Local-only Signal Desktop database decryption
- Two data loading methods: browser upload or volume mount
- Conversation browser with message pagination
- Full-text search across messages
- Attachment viewer
- Conversation statistics page
- Docker containerized deployment
- MIT license

### Platform Support

- Linux: Tested
- macOS: Untested
- Windows: Untested
